# public/

Статические ассеты, отдаются по корню сайта (`/favicon.svg`, `/images/...`, `/video/...`).

Всё подключено через компонент `components/ui/Media.tsx`: пока файла нет — рисуется
аккуратная заглушка с подписью; как только положишь файл по нужному пути — он
автоматически подхватится (без правок кода).

## Куда что класть

| Путь | Что | Размер / формат |
|---|---|---|
| `video/hero-app.mp4` | Видео-демо приложения в hero (fallback — анимированное дерево) | mp4/webm, ~16:10, без звука, зацикленное |
| `images/blog/*.jpg` | Обложки карточек блога | 16:9, ≥1200×675 |
| `images/logos/loop-wallet.svg` | Лого для pull-quote | SVG/PNG, моно |
| `images/partners/*.svg` | Лого партнёров в logo wall | SVG, моно, высота ~32px |
| `favicon.svg` | Иконка вкладки | уже есть |

Имена файлов берутся из `src` в компонентах — см. слаги в коде секций.
