# Палитра Silvana

**Решение (обновлено снова): плоский near-black, без glass и без glow.** Возврат к дисциплине контент-спека («flat, no gradients, no glow») — она читается как осознанный редакторский дизайн, а не шаблон. Поверхности — плоские, заданы hairline-границами и тоном (`.panel`/`.panel-2` в globals.css), без backdrop-blur и без свечения. Sticky-навбар — единственная поверхность с лёгким blur (`.nav-bar`), это стандарт, а не «стекло». Ambient-градиенты и `.gradient-text` удалены. Типографика — Geist + Geist Mono (стенд-ин под Whyte; `.display` = bold, −2.5% трекинг, sentence case). Один тёмный режим.

## Дисциплина цвета (жёсткие правила из контента)
- **magenta `#d6448f`** (родной цвет логотипа, выбран как единый бренд-акцент; hover `#c13a7f`) — ТОЛЬКО основные CTA и live-статусы. Больше нигде.
- **teal `#14b8a6`** — ТОЛЬКО акценты в коде и данных (синтаксис, цифры, чипы данных).
- Всё остальное — плоский near-black фон + off-white текст.
- **Без градиентов. Без glow.** Плоские поверхности.

## Токены (тёмная тема)

| Роль | Значение | Применение |
|---|---|---|
| **bg / base** | `#0B0B0E` | основной фон (near-black, нейтральный холодный) |
| **surface** | `#121217` | карточки, поднятые поверхности (на тон светлее) |
| **surface-2** | `#1A1A20` | вложенные/hover поверхности |
| **fg** | `#ECECEE` | основной текст (off-white, не чистый белый) |
| **muted** | `#86868F` | вторичный текст, подписи, eyebrow-метки |
| **border / line** | `#23232A` | hairline-границы, разделители, рамки mini-UI |
| **accent (magenta)** | `#d6448f` | CTA, live-статусы — и только (hover `#c2387e`) |
| **accent-data (teal)** | `#2bd4b5` | код, цифры, данные — и только |

> teal `#14b8a6` — мой подбор; заменить, если заказчик даст точный HEX.
> Реализация: CSS-переменные в `globals.css`, один тёмный режим.

## Типографика
- Headlines: **Whyte Bold**, letter-spacing −3%, sentence case (из HomePage.md).
- Уточнить лицензию/наличие Whyte; иначе подобрать близкий гротеск (напр. возможный fallback).
- Mono: для тикеров, кода, данных (teal-акцент).

## Контраст / a11y
- Минимум 4.5:1 (fg на bg — ок).
- Видимые teal focus-ring на всех интерактивных элементах.
- Все анимации (marquee, ticker, count-up, draw-on) уважают `prefers-reduced-motion`.
